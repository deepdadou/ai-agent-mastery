**网址：** https://gitee.com

**步骤：**
1. 登录/注册 Gitee 账号
2. 点击 "+" → "新建仓库"
3. 仓库名：`ai-agent-mastery`
4. 描述：`AI 智能体操作秘籍 - 让 AI 为你打工！`
5. 开源协议：MIT
6. 创建后，本地执行：
```bash
cd /home/admin/.openclaw/workspace/skills/ai-agent-mastery
git remote add gitee https://gitee.com/你的用户名/ai-agent-mastery.git
git push gitee main
```
